Enter Grammar (BNF Form)

Instructions

Enter your grammar in BNF format. Each rule should be on a new line.

  • Use `->` for production rules
  • Use `|` for alternatives
  • Example: 
    E -> E + T | T
T -> T * F | F
F -> ( E ) | id
